# 健身健身小程序 - UI 预览效果

## 📱 整体设计风格

### 配色方案
- **主色调**: `#10B981` (翠绿色) - 健康、活力
- **辅助色**: `#F5F5F5` (浅灰) - 背景、卡片
- **文字色**:
  - 主文字: `#1F2937` (深灰)
  - 次要文字: `#6B7280` (中灰)
  - 提示文字: `#9CA3AF` (浅灰)
- **状态色**:
  - 成功: `#10B981` (绿色)
  - 进行中: `#3B82F6` (蓝色)
  - 已完成: `#10B981` (绿色)
  - 已取消: `#EF4444` (红色)

### 组件样式
- **圆角**: 20rpx (卡片)
- **卡片内边距**: 30rpx
- **页面边距**: 30rpx
- **按钮高度**: 90rpx
- **卡片阴影**: 轻微阴影效果

## 🎨 动效与交互

### 签到动画
1. 点击签到按钮
2. 按钮缩放效果（scale 1.2 → 1.0）
3. 显示"✅ 已签到"
4. 数字滚动动画（0 → 5）

### 页面切换
1. 点击 TabBar 切换页面
2. 淡入淡出效果（opacity 0 → 1）
3. 图标切换（灰色 → 绿色）

### 列表加载
1. 首次加载显示骨架屏
2. 数据加载完成后淡入列表项
3. 滚动到底部触发加载更多（如有）

### 按钮点击
1. 按下：scale 0.95
2. 抬起：scale 1.0
3. 点击反馈：轻微震动（小程序端）

## 🎯 设计亮点总结

### ✨ 用户体验
1. **简洁清晰**: 卡片式布局，信息层次分明
2. **视觉引导**: 绿色主色调，健康活力
3. **交互反馈**: 按钮点击、页面切换有动画反馈
4. **信息丰富**: 图标+文字，信息一目了然

### 💡 功能亮点
1. **AI 智能查询**: 数据库没有时自动调用 LLM
2. **签到系统**: 连续签到、日历展示
3. **目标管理**: 日/周/月目标，进度追踪
4. **训练计划**: 模板推荐 + 自定义创建

### 🎨 设计风格
1. **现代简约**: 扁平化设计，圆角卡片
2. **色彩搭配**: 绿色为主，灰度辅助
3. **图标系统**: 统一的图标风格
4. **响应式**: 适配不同屏幕尺寸
